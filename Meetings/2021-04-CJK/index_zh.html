会议目标

MiniApp 是运行在客户端基于 Web 技术（尤其是 CSS 与 Javascript）结合了原生应用技术实现的一种新型移动应用。为了增强不同 MiniApp 平台间的互通性，主流 MiniApp 厂商通过在 MiniApp 工作组（MiniApps WG）和 MiniApp 生态社区组（MiniApps CG）的合作，孵化了与 MiniApp 功能相关的技术提案，旨在促进 MiniApp 生态的互操作性和鲁棒性（见参考资料）。

本次会议的主要目的是汇集中日韩相关厂商，更深入地了解各区域的 MiniApp 生态，介绍 MiniApps WG/CG 当前开展的工作，以及未来 MiniApp 在 W3C 的标准化方向。

本次会议由提前预录的讲者报告（提供中英日韩字幕）以及实时的线上讨论环节（提供中日韩到英文同传）组成。

关于 W3C

W3C 是一个世界范围内开放的标准组织，汇集众多组织与社区成员共同围绕现有和新兴技术展开富有成效的讨论，其为 Web 标准设立了免版税的专利框架。W3C 根据成员和社区输出的优先级开展相应标准化工作。

W3C 是一个开放包容的组织，专注于高效的讨论以及富有成效的行动。 我们的职业道德与行为准则确保我们积极听取来自各方的意见与声音。
